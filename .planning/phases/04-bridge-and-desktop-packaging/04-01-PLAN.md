       - Store in `_server`, `_thread`, `_current_port`. Return actual port.

    5. **`stop_server() -> None`**:
       - If `_server is not None`: set `_server.should_exit = True`, call `_thread.join(timeout=5)`.
       - Reset `_server = None`, `_thread = None`, `_current_port = None`.

    6. **`is_running() -> bool`**:
       - Return `_server is not None and _server.started`.

    7. **`get_port() -> int | None`**:
