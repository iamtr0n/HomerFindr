    **1. Create `packaging/HomerFindr.command`** (macOS double-click launcher):
    ```bash
    #!/bin/bash
    # HomerFindr.command — Double-click to launch HomerFindr CLI in Terminal.
    # Make executable: chmod +x HomerFindr.command

    # Ensure pipx bin directory is in PATH (handles fresh Terminal sessions)
    export PATH="$HOME/.local/bin:$PATH"

    # Source shell profile for any additional PATH entries
    [ -f "$HOME/.zshrc" ] && source "$HOME/.zshrc" 2>/dev/null
    [ -f "$HOME/.bashrc" ] && source "$HOME/.bashrc" 2>/dev/null

    if command -v homerfindr &>/dev/null; then
        homerfindr
    else
        echo ""
        echo "HomerFindr is not installed."
        echo ""
        echo "Install with:"
        echo "  pipx install /path/to/HomerFindr"
        echo ""
        echo "Then run:"
        echo "  pipx ensurepath"
        echo ""
        echo "Press Enter to close."
        read
    fi
    ```

    Set file permissions: `chmod +x packaging/HomerFindr.command`

    **2. Create `packaging/homerfindr_launcher.sh`** (Platypus wrapper for .app bundle):
    ```bash
    #!/bin/bash
    # homerfindr_launcher.sh — Script for Platypus to wrap as macOS .app
    # Platypus config: Interface: None (this script opens Terminal itself)
    #
    # Build .app with Platypus CLI:
    #   /usr/local/bin/platypus \
    #     --name "HomerFindr" \
    #     --interface-type None \
    #     --author "HomerFindr" \
    #     --bundle-identifier "com.homerfindr.app" \
    #     --overwrite \
    #     packaging/homerfindr_launcher.sh \
    #     packaging/HomerFindr.app
    #
    # Then copy to Applications:
    #   cp -r packaging/HomerFindr.app /Applications/

    # Ensure pipx bin is in PATH
    export PATH="$HOME/.local/bin:$PATH"

    # Open a new Terminal window running homerfindr
    osascript -e 'tell application "Terminal" to do script "export PATH=\"$HOME/.local/bin:$PATH\"; homerfindr"'
    ```

    Set file permissions: `chmod +x packaging/homerfindr_launcher.sh`

    **3. Create `packaging/README.md`** with installation and packaging instructions:

    Sections to include:
    - **Install with pipx**: `pipx install .` from project root, then `pipx ensurepath` if needed. Verify with `homerfindr --help` and `which homerfindr`.
    - **Development install**: `pipx install --editable .` for live source changes.
    - **macOS Launcher (.command)**: Copy `HomerFindr.command` anywhere (Desktop, Applications), double-click to open Terminal with homerfindr. Note: must `chmod +x` if not already.
    - **macOS App Bundle (Platypus)**: Optional polished .app for Dock. Requires Platypus 5.5+ with CLI tool enabled. Provide the exact platypus CLI command from the research. Note that the .app opens Terminal — it is a launcher, not a host (Platypus cannot provide interactive stdin).
    - **Troubleshooting**: "command not found" after install -> run `pipx ensurepath`, restart terminal. Port already in use -> HomerFindr auto-selects next available port (8000-8010).
