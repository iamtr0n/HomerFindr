Merge loaded data over a copy of DEFAULT_CONFIG using a shallow merge per top-level key (so new keys added in future versions auto-populate):
```python
import copy
merged = copy.deepcopy(DEFAULT_CONFIG)
for key in merged:
    if key in data and isinstance(data[key], dict):
        merged[key].update(data[key])
    elif key in data:
        merged[key] = data[key]
return merged
```

2. `run_smtp_wizard(existing: dict | None = None) -> dict | None`:
   - Provider select via `questionary.select("Email provider:", choices=["Gmail", "Outlook/Hotmail", "Yahoo", "Custom"], style=HOUSE_STYLE)`
   - After Gmail is selected, print dim hint: `[dim]Gmail tip: Use an App Password from myaccount.google.com/apppasswords[/dim]`
   - Pre-fill server and port from PROVIDER_PRESETS
   - `questionary.text("SMTP server:", default=preset["server"], style=HOUSE_STYLE)`
   - `questionary.text("Port:", default=str(preset["port"]), style=HOUSE_STYLE)` — convert to int with try/except, default 587 on ValueError
   - `questionary.text("Your email address:", default=existing.get("email", ""), style=HOUSE_STYLE)`
   - `questionary.password("App password (input masked):", style=HOUSE_STYLE)` — per D-02 masked input
   - If any `.ask()` returns None (Ctrl+C), return None immediately
   - Return dict: `{"provider": provider, "server": server, "port": int(port_str), "email": email, "password": password, "recipients": existing.get("recipients", [])}`

3. `test_smtp(smtp_cfg: dict) -> bool`:
   - Uses `smtplib.SMTP(smtp_cfg["server"], smtp_cfg["port"], timeout=10)` with `starttls()` and `login()`
   - Sends a test email (MIMEText) to `smtp_cfg["email"]` with subject "HomerFindr: SMTP Test"
   - On success: `console.print(Panel("[bold green]Test email sent successfully![/bold green]", border_style="green"))`; return True
   - On failure: `console.print(f"[red]Test failed: {e}[/red]")`; return False

4. `run_smtp_wizard_with_test(existing: dict | None = None) -> dict | None`:
   - Calls `run_smtp_wizard(existing)` to collect credentials
   - If None returned, return None
   - Calls `test_smtp(result)`
   - If test passes, return result
   - If test fails (per D-04): offer `questionary.select("What would you like to do?", choices=["Retry settings", "Save anyway (skip test)"], style=HOUSE_STYLE)`
   - "Retry settings" -> call `run_smtp_wizard_with_test(existing)` recursively
   - "Save anyway" -> return the result as-is

Import `console` and `HOUSE_STYLE` from `homesearch.tui.styles`. Import `smtplib`, `email.mime.text.MIMEText` from stdlib. Import `Panel` from `rich.panel`.
