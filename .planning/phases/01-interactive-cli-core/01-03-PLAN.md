Create `homesearch/tui/wizard.py` implementing all 15 fields per D-08 through D-12.

**Field order (per D-09):** Type, Property, Location, Radius, ZIP Discovery, Price, Beds, Baths, Sqft, Lot, Year, Floors, Basement, Garage, HOA

**Every `.ask()` call must check for None (Ctrl+C returns None). If None, return None from the wizard.**

**Field implementations:**

1. **Listing Type** — `questionary.select`: choices `["For Sale", "For Rent", "Recently Sold"]`. Map: `{"For Sale": ListingType.SALE, "For Rent": ListingType.RENT, "Recently Sold": ListingType.SOLD}`

2. **Property Type** — `questionary.select`: choices `["Any", "Single Family", "Condo", "Townhouse", "Multi-Family", "Commercial", "Land"]`. "Any" maps to empty list `[]`. Others map to `[PropertyType.SINGLE_FAMILY]` etc.

3. **Location** — `questionary.text` (per D-08, the ONLY free-typing field): prompt "City, State or ZIP code:". Validate non-empty. If empty or None, return None.

4. **Radius** — `questionary.select`: choices `["5 miles", "10 miles", "25 miles", "50 miles", "100 miles"]`, default `"25 miles"`. Parse int from first word.

5. **ZIP Discovery** — After getting location+radius, run ZIP discovery with a Rich spinner:
   - Use `console.status("Discovering ZIP codes...")` (NOT Rich Live, since no questionary prompt is active during this)
   - Call `discover_zip_codes(location, radius)`
   - If ZIPs found, display count and use `questionary.checkbox` with all ZIPs pre-checked. User unchecks to exclude (per Research Open Question 1 recommendation). Cap display at 30 ZIPs.
   - `zip_codes` = checked ZIPs, `excluded_zips` = unchecked ZIPs

6. **Price Range** — `questionary.select` (per D-11): choices `["Any", "Under $100k", "$100k - $200k", "$200k - $300k", "$300k - $400k", "$400k - $500k", "$500k - $750k", "$750k - $1M", "$1M - $1.5M", "Over $1.5M"]`. Parse with `_parse_price_range()` helper returning `(price_min, price_max)`. "Any" returns `(None, None)`.

7. **Beds** — `questionary.select`: choices `["Any", "1+", "2+", "3+", "4+", "5+"]`. "Any" = None, else `int(choice[0])`.

8. **Baths** — `questionary.select`: choices `["Any", "1+", "1.5+", "2+", "2.5+", "3+"]`. "Any" = None, else `float(choice.rstrip("+"))`.

9. **Sqft Range** — `questionary.select` (per D-11): choices `["Any", "Under 1,000", "1,000 - 1,500", "1,500 - 2,000", "2,000 - 3,000", "3,000 - 4,000", "Over 4,000"]`. Parse with `_parse_sqft_range()` returning `(sqft_min, sqft_max)`.

10. **Lot Size** — `questionary.select` (per D-11): choices `["Any", "Under 5,000 sqft", "5,000 - 10,000 sqft", "10,000 - 20,000 sqft", "Over 20,000 sqft", "Over 1 acre"]`. Parse with `_parse_lot_range()`.

11. **Year Built** — `questionary.select`: choices `["Any", "2020+", "2010+", "2000+", "1990+", "1980+", "1970 or older"]`. Parse with `_parse_year()` returning `year_built_min` (or None for "Any").

12. **Floors/Stories** — `questionary.select`: choices `["Any", "1+", "2+", "3+"]`. "Any" = None, else `int(choice[0])`.

13. **Basement** — `questionary.select`: choices `["Don't care", "Must have", "No basement"]`. Map: `{"Must have": True, "No basement": False, "Don't care": None}`.

14. **Garage** — `questionary.select`: choices `["Don't care", "Must have", "No garage"]`. Same mapping as basement. If "Must have", follow up with garage spaces select: `["Any", "1+", "2+", "3+"]`.

15. **HOA** — `questionary.select`: choices `["Any / No limit", "No HOA ($0)", "Up to $100/mo", "Up to $200/mo", "Up to $300/mo", "Up to $500/mo"]`. Parse with `_parse_hoa()`.

- "Yes" returns the SearchCriteria

- "Cancel" returns None
