Create `homesearch/tui/saved_browser.py` implementing the full saved searches browser (per D-11, D-12, D-13, D-14, D-15).

Module docstring: `"""Saved searches browser — view, run, toggle, rename, and delete."""`

Imports: `questionary` from questionary, `Table` from rich.table, `Panel` from rich.panel, `console` and `HOUSE_STYLE` from `homesearch.tui.styles`, `datetime` from datetime. Lazy imports inside functions for `database` and `results` modules to avoid circular imports.

**1. `show_saved_searches_browser() -> None`** (main entry point):
   - `from homesearch import database as db`
   - `db.init_db()`
   - Loop (so user returns to list after each action):
     - `searches = db.get_saved_searches()`
     - If not searches: print `console.print("[yellow]No saved searches yet. Run a search from the main menu to save one.[/yellow]")` per D-15. Return.
     - Call `_render_searches_table(searches)` — renders Rich Table BEFORE questionary fires (Live-exits-first rule)
     - Build choices: `["\u2190 Back"] + [f"{s.name} ({s.criteria.location or 'N/A'})" for s in searches]`
     - `pick = questionary.select("Select a search:", choices=choices, style=HOUSE_STYLE).ask()`
     - If pick is None or pick == "\u2190 Back": return
     - Find index: `idx = choices.index(pick) - 1` (offset for Back entry)
     - Call `_show_search_submenu(searches[idx])`
     - After submenu returns, loop continues (re-fetches and re-renders table)

**2. `_render_searches_table(searches: list) -> None`** (per D-11):
   - Create `Table(show_header=True, header_style="bold", show_lines=True)`
   - Columns: `"Name"` (style="white", min_width=20), `"Location"` (style="cyan", min_width=16), `"Last Run"` (style="dim", width=20), `"Active"` (justify="center", width=8)
   - For each search:
     - `last_run = s.last_run_at.strftime("%Y-%m-%d %H:%M") if s.last_run_at else "Never"`
     - `active = "[green]\u2713[/green]" if s.is_active else "[red]\u2717[/red]"`
     - `table.add_row(s.name, s.criteria.location or "N/A", last_run, active)`
   - `console.print(table)`

**3. `_show_search_submenu(search) -> None`** (per D-12):
   - `choices = ["\u2190 Back", "Run Now", "Toggle Active/Inactive", "Rename", "Delete"]`
   - `action = questionary.select(f"Action for: {search.name}", choices=choices, style=HOUSE_STYLE).ask()`
   - Dispatch:
     - None or Back -> return
     - "Run Now" -> `_run_search_now(search)` (per D-14)
     - "Toggle Active/Inactive" -> `_toggle_active(search)`
     - "Rename" -> `_rename_search(search)`
     - "Delete" -> `_delete_search(search)`

**4. `_run_search_now(search) -> None`** (per D-14):
   - `from homesearch.tui.results import execute_search_with_spinner, display_results`
   - `from homesearch import database as db`
   - `console.print(f"[cyan]Running: {search.name}[/cyan]")`
   - `results = execute_search_with_spinner(search.criteria)`
   - `display_results(results, search.criteria)`
   - Update last_run_at: `db.update_search(search.id, last_run_at=datetime.now().isoformat())`

**5. `_toggle_active(search) -> None`**:
   - `from homesearch import database as db`
   - `new_state = not search.is_active`
   - `db.update_search(search.id, is_active=new_state)`
   - `label = "active" if new_state else "inactive"`
   - `console.print(f"[green]'{search.name}' is now {label}.[/green]")`

**6. `_rename_search(search) -> None`**:
   - `new_name = questionary.text("New name:", default=search.name, style=HOUSE_STYLE).ask()`
   - If new_name and new_name.strip():
     - `from homesearch import database as db`
     - `db.update_search(search.id, name=new_name.strip())`
     - `console.print(f"[green]Renamed to '{new_name.strip()}'.[/green]")`

**7. `_delete_search(search) -> None`** (per D-13):
   - `confirmed = questionary.confirm(f"Delete '{search.name}'? This cannot be undone.", default=False, style=HOUSE_STYLE).ask()`
   - If confirmed:
     - `from homesearch import database as db`
     - `db.delete_search(search.id)`
     - `console.print(f"[green]Deleted '{search.name}'.[/green]")`
   - Else: `console.print("[dim]Cancelled.[/dim]")`
